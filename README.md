# SOC Investigations & Incident Response Portfolio

A collection of incident response triage reports and SOC investigation exercises. Each report documents real-world attack simulations analyzed through PCAP forensics, SIEM log correlation, and threat intelligence. Work includes full IR documentation covering scope, impact, timeline, MITRE ATT&CK mapping, and remediation recommendations.

---

## Tools & Skills
- **Network Forensics:** Wireshark, PCAP analysis
- **SIEM:** Splunk (log correlation, query building, event analysis)
- **Threat Intelligence:** MITRE ATT&CK framework, IOC extraction
- **Documentation:** IR triage reports, incident timelines, evidence mapping
- **Security Concepts:** Brute-force detection, credential compromise, lateral movement, post-authentication exploitation
